## Installing

A step by step series of examples that tell you how to start the app

Download the proyect

```
git clone https://github.com/Vente16/tasksApp.git
```

Get in the project

```
cd tasksApp
```
Install dependencies

```
npm install  
```

Run the app

```
npm start 
```

Or
```
npm i yarn 
yarn start 
```
